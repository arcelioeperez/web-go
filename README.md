# Writing Web Applications 

Git clone, build and run:  
```
git clone git@github.com:arcelioeperez/web-go.git && cd web-go/
chmod +x build.sh
./build.sh
```

Or:  
```
cd gowiki && go build wiki.go
./wiki
```

Then go to: http://localhost:8080/view/ANewPage
